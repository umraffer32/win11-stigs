## What is the STIG ID?
**WN11-AC-000035** — Passwords must, at a minimum, be 14 characters.

Password length is the single biggest factor in resistance to offline cracking. Short NTLM hashes can be cracked in seconds by modern GPU rigs. 14 characters meaningfully raises the bar against brute force and rainbow table attacks.

## How we are fixing it
Unlike registry-based STIGs, account policy is set via Windows' built-in `net accounts` command. This modifies the local SAM (Security Account Manager) policy on standalone machines.

| Setting | Value |
|---|---|
| Tool | `net accounts` |
| Parameter | `/minpwlen:14` |

**PowerShell (run as Administrator):**

```powershell
# Set minimum password length to 14
net accounts /minpwlen:14

# Local verify
net accounts
```

The `net accounts` output should display:
```
Minimum password length:                              14
```

> Note: This applies to future password changes only. The user's current password is not invalidated, and Nessus checks the policy itself — not the length of the active password.
